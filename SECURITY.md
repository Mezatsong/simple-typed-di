# Security Policy

## Supported Versions

Only the latest version of `simple-typed-di` is supported with security updates.

| Version | Supported |
| ------- | --------- |
| 1.x.x   | ✅ Yes    |
| < 1.0.0 | ❌ No     |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please **do not open a public issue**. Instead, please report it privately by emailing **meztsacar@gmail.com**.

We will acknowledge your email within 48 hours and provide a timeline for a fix if necessary.
